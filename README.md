# AudioWaveformGraph for iOS and Mac
## Description

AudioWaveformGraph is the companion project to the [Drawing Big Graphs with Swift andÂ UIKit](https://blog.madebywindmill.com/drawing-big-graphs-with-swift-and-uikit-167f48c3967a) tutorial.
